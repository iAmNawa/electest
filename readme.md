#ELECTRON APP

creating a basic electron app from the electron docs.

your-app/  
  ├── package.json  
  ├── main.js  
  └── index.html  


MacOS / Linux  
type npm i in the console and then ./node_modules/.bin/electron .

Windows  
$ .\node_modules\.bin\electron .

Node v8.2.0 and later  
npx electron .
